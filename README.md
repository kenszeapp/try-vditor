# try-vditor
Show how to apply a vditor works with Vue3 JS. 


## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```


### Others 
This created by ```npm create vue@3``` template
